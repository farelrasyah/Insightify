CARA MEMBUAT IKON EKSTENSI
============================

File ini berisi instruksi untuk membuat ikon ekstensi Insightify.

UKURAN YANG DIPERLUKAN:
- icon16.png (16x16 px) - Toolbar
- icon32.png (32x32 px) - Toolbar Retina
- icon48.png (48x48 px) - Extensions page
- icon128.png (128x128 px) - Chrome Web Store

TOOLS YANG BISA DIGUNAKAN:
1. Online: https://favicon.io/
2. Online: https://realfavicongenerator.net/
3. Software: Photoshop, GIMP, Figma
4. Command line: ImageMagick

MENGGUNAKAN SVG KE PNG:
1. Buka icon.svg di browser
2. Screenshot atau export sebagai PNG
3. Resize ke ukuran yang diperlukan
4. Atau gunakan online converter

DESIGN GUIDELINES:
- Background: Gradient blue (#667eea to #764ba2)
- Icon: Brain atau lightbulb putih
- Style: Modern, minimal, memorable
- Contrast: Pastikan terlihat di light/dark theme
